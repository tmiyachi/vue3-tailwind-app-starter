# vue-app-starter

Vue.js を用いた web アプリ開発のための個人用スターター

## Description

- Vite によるバンドル
- ESlint，stylelint による静的コード解析
- Prettier によるフォーマット
- tailwindcss の利用

## Usage

Install package

```bash
npm install
```

build

```bash
npm run build
```

lint

```bash
npm run lint
```

fix

```bash
npm run fix
```

dev

```bash
npm start
```
