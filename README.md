# vue-app-starter

Vue.js を用いた web アプリ開発のための個人用スターター

## Description

- webpack によるバンドル
- ESlint，stylelint による静的コード解析
- Prettier によるフォーマット
- Babel によるトランスコンパイル
- tailwindcss の利用

## Note

[eslint-plugin-vue](https://github.com/vuejs/eslint-plugin-vue) は最新バージョンを使用しVue3準拠としている．

## Usage

Install package

```bash
npm install
```

build

```bash
npm run build
```

lint

```bash
npm run lint
```

fix

```bash
npm run fix
```

dev

```bash
npm start
```
